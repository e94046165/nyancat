# Nyancat CLI

Nyancat rendered in your terminal.

[![Nyancats](http://nyancat.dakko.us/nyancat.png)](http://nyancat.dakko.us/nyancat.png)

## Setup

First build the C application:

    make

You can run the C application standalone.

    ./nyancat
